# Sustainable Aviation Fuel (SAF) Premium Price Analysis

## Overview

This project analyses Sustainable Aviation Fuel (SAF) pricing data to explore premium trends and generate business insights using Python. 
The analysis includes data cleaning, exploratory data analysis and visualisation to better understand pricing patterns within the 
sustainable aviation industry.

---

## Objective

The goal of this project is to:

- Explore Sustainable Aviation Fuel pricing data
- Identify price premium trends
- Clean and prepare raw data for analysis
- Create visualisations that communicate key findings
- Develop practical data analytics skills using Python

---

## Tools & Technologies used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Excel

---

## Project Workflow

1. Imported and explored the dataset
2. Cleaned and prepared the data
3. Performed exploratory data analysis (EDA)
4. Created visualisations
5. Interpreted results and documented insights

---

## Key Insights

- Identified trends in Sustainable Aviation Fuel pricing.
- Explored variations in fuel price premiums across the dataset.
- Demonstrated how data visualisation can support business understanding and decision-making.
